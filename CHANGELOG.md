# Pterodactyl Changelog

## [Initial Release] - {PR_MERGE_DATE}

### Features
- **Server List**: View all servers with real-time status (Running, Offline, etc.).
- **Power Actions**: Control server state (Start, Stop, Restart, Kill) with confirmation dialogs.
- **Interactive Console**: Stream logs and send commands directly to your server via WebSocket.
- **Real-time Monitoring**: Visualize CPU, Memory, Disk, and Network usage with ASCII charts.
- **Secure**: API keys are stored safely using Raycast's encrypted preferences.
